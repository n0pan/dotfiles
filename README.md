## Installation

This repo contains configuration files that can be used to setup a new computer running macOS. 

```bash
git clone git@github.com:n0pan/dotfiles.git ~/dotfiles
sh ~/dotfiles/install.sh
vim +PackerSync +qall
vim +UpdateRemotePlugins +qall
```
